# Schedule_Management_App
Ứng dụng quản lý công việc học tập cho sinh viên (Quản lý thời gian, tiến độ, danh sách công việc theo cá nhân/nhóm)

● tạo công việc cần làm,
● gán người phụ trách,
● đặt ngày bắt đầu, hạn chót.
● đánh dấu trạng thái hoàn thành, ghi chú mức độ hoàn thành
● xem danh sách công việc theo cá nhân hoặc theo nhóm.
